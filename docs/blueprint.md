# **App Name**: LearnAI Academy

## Core Features:

- User Authentication: Secure user authentication with Firebase Auth, supporting email, Google, GitHub, and Microsoft login.
- AI Quiz Generator: Generate custom quizzes from any topic/document using the Gemini API, adjusting difficulty based on topic complexity.
- Smart Flashcards: AI-created flashcards with a spaced repetition algorithm to optimize learning.  Users can upload information, and AI generates flashcards to the provided input.
- Personalized Dashboard: A customizable dashboard with widgets displaying daily goals, upcoming sessions, recent activity, and AI-powered study recommendations.
- Data Storage: Store user data and quiz information in Firestore

## Style Guidelines:

- Primary color: Indigo gradient (#4F46E5 to #7C3AED) for a modern and intelligent feel.
- Background color: Light gray (#FAFAF9), providing a clean and warm backdrop.
- Accent color: Emerald (#10B981) to highlight important actions and provide a sense of growth and progress.
- Headline font: 'Inter', a grotesque-style sans-serif, to convey a modern, neutral, objective look. It will be used for headers to ensure excellent readability on primary navigation elements.
- Body font: 'PT Sans' will complement Inter in order to deliver blocks of easy to read content throughout the app.
- Use a consistent style of outline icons to maintain a clean and modern aesthetic.
- Implement a responsive layout using CSS Grid and Flexbox to ensure the application is accessible on various devices.
- Use subtle animations with Framer Motion to enhance user experience, such as page transitions, hover effects, and loading states.
